# 12a - Webhooks

This project emulates a webhook system for a dorm. It will have the following
events

- Users
  - User created
  - User update payments
  - User update has_late_payments
  - User update role
  - User deleted
- Rooms
  - Room created
  - Room update tenant
  - Room update rent
  - Room deleted
- Complaints
  - Complaint created
  - Complaint update is_resolved
  - Complaint deleted
